# IELTS Speaking Dataset v1.0

**Дата создания:** 2024  
**Статус:** Зафиксированная версия

## Описание

Первая стабильная версия датасета после Synthetic Expansion и Topic Expansion.

## Статистика

- **Всего ответов:** 2522
- **Part 1:** 948 ответов (37.6%)
- **Part 2:** 748 ответов (29.7%)
- **Part 3:** 826 ответов (32.8%)

## Известные проблемы

1. **Шаблонность генерации** - ответы слишком похожи друг на друга
2. **Субскоры не связаны с текстом** - GRA низкий, но нет грамматических ошибок
3. **Извлечение темы из вопросов** - работает плохо для некоторых типов вопросов
4. **Quality flag логика** - слишком грубая

## Типы источников

- `synthetic`: 1946 ответов (77.2%)
- `synthetic_augmented`: 527 ответов (20.9%)

## Использование

Эта версия используется как baseline для сравнения с улучшенными версиями.

**Не изменять!** Все изменения должны идти в новые версии.
